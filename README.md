# CrackIt.AI

CrackIt.AI is a MERN + AI/ML-based placement guidance platform designed to help users prepare for placements effectively. It combines modern web technologies with AI-driven insights to provide personalized roadmaps, mock tests, and real-time chatrooms for company-specific discussions.

## Features

- **Home Page**: Landing page with a modern UI, dark/light mode toggle, and navigation options.
- **Authentication**: Secure registration and login with JWT and bcrypt, password strength validation, and user settings.
- **User Goal Setup**: Form to set target companies, preferred domain, tech stack, and expected LPA.
- **Skill Survey**: Capture knowledge levels in DSA, Core CS, and development experience.
- **AI Roadmap Generator**: Personalized study plans and progress tracking.
- **Mock/Practice Tests**: Topic-wise quizzes with AI feedback.
- **Company-Wise Chatrooms**: Real-time chat with Socket.io.
- **Company QnA Vault**: Fetch and display latest interview QnAs.
- **Dashboard**: Personalized dashboard with progress, tasks, and quick access to features.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS, React Router, Socket.io-client.
- **Backend**: Node.js, Express.js, MongoDB, JWT, bcrypt, Socket.io.
- **AI Microservice**: Python, Scikit-learn, Rule-based roadmap generator.
- **Database**: MongoDB.

## Getting Started

1. Clone the repository.
2. Install dependencies for both frontend and backend.
3. Set up environment variables.
4. Run the development servers.

## Folder Structure

- `frontend/`: React.js application.
- `backend/`: Node.js and Express.js server.
- `ai_service/`: Python-based AI microservice.
- `database/`: MongoDB collections and schemas.
